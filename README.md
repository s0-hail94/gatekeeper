A simple guide to demonstrate the different authentication methods in Django Rest Framework

## Installation

Simply run:

```
$ ./install.sh
```

## Running

``` shell
$ source env/bin/activate
$ python3 manage.py makemigrations
$ python3 manage.py migrate
$ python3 manage.py runserver
```
